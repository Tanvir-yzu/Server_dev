Quick Start Guide
=================

Getting Started
---------------

After installation, you can access the application at http://127.0.0.1:8000/

Admin Interface
---------------

Access the Django admin at http://127.0.0.1:8000/admin/ using your superuser credentials.

Key Features
------------

* **Authentication**: User registration, login, and profile management
* **DevOps**: Project management and deployment tracking
* **Collaboration**: Team collaboration and invitation system
* **System**: Health monitoring and log management

API Endpoints
-------------

The application provides various API  endpoints for different modules:

* ``/auth/`` - Authentication related endpoints
* ``/collaboration/`` - Collaboration features
* ``/system/`` - System monitoring
